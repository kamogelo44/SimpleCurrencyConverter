# Simple Currency Converter

A Java console application for converting amounts between different currencies using hardcoded exchange rates.

## What This Program Does

- Convert amounts between 12 different currencies
- Supports world super powers and African currencies
- Displays available currency codes
- Provides real-time conversion calculations
- Uses USD as base currency for all conversions

## How to Run

1. Make sure you have Java installed
2. Open the project in your preferred code editor
3. Locate and run the SimpleCurrencyConverter.java file
4. Follow the currency conversion instructions on screen

## Sample Usage
=== SIMPLE CURRENCY CONVERTER ===

=== AVAILABLE CURRENCIES ===
USD - US Dollar

RUB - Russian Ruble
INR - Indian Rupee
ZAR - South African Rand
NGN - Nigerian Naira
EGP - Egyptian Pound

GHS - Ghanaian Cedi
============================

Enter amount to convert: 100
Enter source currency code (e.g., USD): USD

100.00 USD = 1480.00 ZAR
